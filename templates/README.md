# Plantillas de composicion (Remotion)

Cada subcarpeta contiene las composiciones/ajustes de exportacion
especificos para una plataforma, todas partiendo del mismo master en
formato vertical 9:16:

- shorts/: composiciones para YouTube Shorts (hasta 60s).
- youtube/: reservado para variantes de video largo, si el canal lo
  necesita en el futuro.
  - tiktok/: ajustes de exportacion y metadata propios de TikTok.
  - instagram/: ajustes de exportacion propios de Instagram Reels.
